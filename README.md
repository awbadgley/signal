# VEKTR

**A browser-based step sequencer with a retro 80s aesthetic.**

Try it live → **[awbadgley.github.io/vektr](https://awbadgley.github.io/vektr/)**

---

## What it is

VEKTR is a 16-step sequencer that runs entirely in your browser — no install, no account, no plugins. Four channels, each with its own synthesized sound, laid out on a grid you can tap out patterns on in seconds.

## Sounds

| Channel | Type | Character |
|---|---|---|
| LEAD | Detuned square wave | Thick, chorused synth lead |
| PULSE | Thin pulse wave | Buzzy SID-chip / arcade tone |
| BASS | Sawtooth + filter sweep | Moog-style synth bass |
| DRUM | Noise + oscillator | Kick, snare, and hi-hat |

## Features

- 16-step grid across 4 channels
- BPM control with tap tempo and swing
- 2-octave piano keyboard for note entry
- Scale-aware highlighting (pentatonic, major, minor, and more)
- Save and load patterns
- Share patterns via URL
- WAV export
- Warp-speed starfield background (obviously)

## How to use

1. Select a channel (LEAD, PULSE, BASS, DRUM)
2. Pick a note on the keyboard
3. Click steps on the grid to activate them
4. Hit PLAY

That's it.

## Running locally

It's a single HTML file. Download `index.html` and open it in any modern browser.
