# VEKTR

**A browser-based step sequencer with a retro 80s aesthetic.**

Try it live → **[awbadgley.github.io/vektr](https://awbadgley.github.io/vektr/)**

---

## What it is

VEKTR is a 16-step sequencer that runs entirely in your browser — no install, no account, no plugins. Four channels, each with its own synthesized sound, laid out on a grid you can tap out patterns on in seconds.

## Sounds

| Channel | Type | Character |
|---|---|---|
| LEAD | Detuned square wave | Thick, chorused synth lead |
| PULSE | Thin pulse wave | Buzzy SID-chip / arcade tone |
| BASS | Sawtooth + filter sweep | Moog-style synth bass |
| DRUM | Noise + oscillator | Kick, snare, and hi-hat |

## Features

- 16-step grid across 4 channels
- **4 patterns (A–D) with song-mode chaining** — build full arrangements, not just loops
- **Accents** — shift-click a step to make it punch harder
- **C64-style arpeggiator** on the PULSE channel
- Tape echo + plate reverb with per-channel send levels
- BPM control with tap tempo and swing
- 2-octave piano keyboard for note entry — or play the bottom row of your computer keyboard (Z–M)
- Scale-aware highlighting (pentatonic, major, minor, and more)
- Save and load full songs
- Share songs via URL
- WAV export — renders the entire chain through the same synth engine you hear live
- **DEMO button** — load a ready-made groove and hit play
- Warp-speed starfield background (obviously)

## How to use

1. Hit **DEMO**, then **PLAY** — that's the fastest way to hear what it does
2. Select a channel (LEAD, PULSE, BASS, DRUM)
3. Pick a note on the keyboard
4. Click steps on the grid to activate them (shift-click for accents)
5. Build patterns A–D, chain them with **+**, flip **SONG: ON**, and hit PLAY

## Keyboard shortcuts

| Key | Action |
|---|---|
| Space | Play / stop |
| Z–M row | Play notes (piano layout) |
| 1–4 | Select channel |
| Shift + 1–4 | Select pattern |
| ↑ / ↓ | Octave up / down |
| ← / → | BPM down / up |
| T | Tap tempo |
| R | Random fill |

## Running locally

It's a single HTML file. Download `index.html` and open it in any modern browser.
